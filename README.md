# Local Folders for NotebookLM

A clean, modern, and 100% private Chrome extension to organize your Google NotebookLM dashboard workspace into custom, color-coded folders.

Because Google NotebookLM does not natively support folders, this extension injects a native-feeling sidebar and tag-based folder management directly into the page DOM.

## Key Features

- **📁 Native-Style Sidebar:** A beautiful sidebar docked on the left of your notebook list that shows default (All, Uncategorized) and custom folders.
- **🏷️ Folder Assignment Tags:** Small tag pills next to each notebook title that open a custom dropdown menu on click to assign folders.
- **⚙️ Custom Folder Management:** Create, rename, delete, color-code, and choose emojis for your folders using a premium popup modal.
- **🔒 100% Private & Local:** All folder structures, colors, and notebook-to-folder maps are saved in your browser's local storage (`chrome.storage.local`). No cloud syncing, no third-party servers, and no data tracking.
- **💾 Backup & Portability:** Import and export your entire workspace organization setup as a JSON file from the extension's popup panel in the toolbar.
- **🎨 Dark & Light Modes:** Fully responsive variables that automatically adapt to Google NotebookLM's theme.

## Installation

As this is a custom extension, you can install it using Chrome's Developer Mode:

1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** using the toggle in the top-right corner.
4. Click **Load unpacked** in the top-left corner.
5. Select the project folder containing the source files (e.g. `d:/projectts/notebooklm folder`).
6. Navigate to [NotebookLM](https://notebooklm.google.com/) to start organizing!

## Privacy Policy
This extension does not collect, store, or transmit any personal data, notebooks, files, or telemetry. All state is maintained locally within the browser.
